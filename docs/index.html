podman-minimal

Star Report issue

One command runs the container. You choose the image; on a typical Linux machine with NVIDIA, the GPU is wired in for you.

Example

$ podman-minimal --image docker.io/nvidia/cuda:12.4.1-base-ubuntu22.04 nvidia-smi -L
GPU 0: NVIDIA GeForce RTX 5080 (UUID: GPU-e611ee94-8aa1-ec0c-cf66-dfb053851429)

Output captured on a Linux host with rootless Podman and an NVIDIA GPU. Your line will match your hardware; without a GPU the same command still runs the image, but nvidia-smi may be empty or fail inside the container.

Install

python3 -c "import urllib.request as u; exec(u.urlopen('https://raw.githubusercontent.com/vincenzoml/podman-minimal/main/podman-minimal.py').read().decode())" --install

Windows: use python instead of python3. If Podman is missing, the launcher asks before using package managers, Homebrew, winget, or choco. Docs: README.
